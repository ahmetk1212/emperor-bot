# emperor-bot

Local-first, modular AI agent skeleton for safe night automation.

## MVP Features
- Night task scheduling
- Policy-based permission checks
- Modular plugin execution
- Audit log and morning summary report
- Paper-trading simulation with strict risk limits

## Safety Notes
- No guaranteed financial profit.
- Trading is simulation-only (paper trading) in MVP.
- High-risk actions should require user approval and kill-switch controls.

## Commands
```bash
npm install
npm run lint
npm run build
npm test
```
